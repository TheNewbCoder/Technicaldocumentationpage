Introduction

HTML is the standard language for building a webpage. HTML stands for Hyper-Text Markup Language. It is the skeleton of the webpage where you add all the information you want to display, be it in text or images. It will display evertything as is without anything fancy added to it.

Elements

HTML elements always starts with an opening tag and almost always ends with a closing tag 

                       
                           <html>
                           <head>
                           </head>
                           <body>
                           <h1> This is the page heading<h1>
                           <p> Here is a very basic structure of a webpage. <p>
                           </body>
                           </html>

Attributes

Attributes provide additional information to a HTML tag. They are always written in the opening tag for an element.

                        ***Example***
                        <a href="thisisanattribute.com"></a>
                        <p class="classattribute"></p>

Above examples shows the href-attribute for linking webpages and the class-attribute to assign a class to an element.

Headings

Headings are the titles in a webpage are defined in different sizes raning from 1 to 6, where 1 is most important. 

                            <h1>Heading 1</h1>
                            <h2>Heading 2</h2>
                            <h3>Heading 3</h3>
                            <h4>Heading 4</h4>
                            <h5>Heading 5</h5>
                            <h6>Heading 6</h6>

Headings are used by search engines to index the structure and content of a webpage.

Pragraphs

Paragraphs are section of text enclosed in <p> tags.

Formatting text

Formatting text is usually done through CSS but can be done with inline html-tags like <strong> which will make the enclosed text bold.
